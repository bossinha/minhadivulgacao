# Minha Divulgação - Portal Multi-Tenant SaaS (GitHub + Firebase)

Este projeto é uma plataforma para aluguel de guia digital e rádio online, agora 100% compatível com **GitHub Pages** e **Firebase**.

## 🚀 Como colocar no ar (GitHub Pages)

1. **GitHub**:
   - Use o botão **"Export to GitHub"** no menu do Google AI Studio.
   - No seu repositório no GitHub, vá em **Settings** > **Pages**.
   - Em "Build and deployment", escolha **GitHub Actions**.
   - O GitHub irá compilar e colocar seu site no ar automaticamente no seu domínio.

2. **Firebase (Banco de Dados)**:
   - O projeto já está configurado para usar o Firebase. 
   - Ao exportar, o arquivo `firebase-applet-config.json` irá junto, garantindo que o banco de dados funcione.
